# 実装計画書: 社会的孤立とベンゾジアゼピン系薬剤処方の関連解析 (Project NDB_001)

**プロジェクトID**: `NDB_001_social_isolation_bz`
**作成日**: 2026年2月12日
**対象データ**: 第10回NDBオープンデータ、2020年国勢調査

## 1. プロジェクトの目的
高齢者におけるベンゾジアゼピン（BZ）系睡眠薬・抗不安薬の処方量（人口あたり）と、社会的孤立指標（独居老人割合等）の地域相関を都道府県単位で検証する。
前回の反省を踏まえ、**NDBオープンデータの「薬効分類別数量」と国勢調査の実数**を用いた、解釈のブレが少ない堅牢な生態学的研究を目指す。

## 3. データ取得・ETL戦略

### 3.1 分子：BZ系薬剤処方量 (NDB)
- **ソース**: 第10回NDBオープンデータ「薬効分類別数量」。
- **対象ファイル**: 内服（入院外・院内/院外）。
- **抽出ロジック**:
    - 薬効分類コード `112` (催眠鎮静剤) および `117` (精神神経用剤) を抽出。
    - さらに、BZ系・非BZ系（Z-drug）を区別するため、一般的な成分名（エチゾラム、ブロチゾラム、ゾルピデム等）または詳細な薬効コードでフィルタリングする機能を持たせる。
    - **重要**: 今回は「数量（錠数等）」を「高齢者人口」で割った値を指標とする。「患者数」への無理な換算は行わない。

### 3.2 分母・説明変数：社会的指標 (e-Stat)
- **ソース**: 2020年国勢調査（e-Stat APIまたはCSVダウンロード）。
- **取得項目**:
    - 都道府県別・年齢階級別（65歳以上）人口。
    - 65歳以上単独世帯数（独居）。
    - 高齢者就業率（社会的参加の代理変数）。

## 4. 解析プロセス

### Phase 1: データセット構築
1.  **BZ抽出**: `01_extract_bz_volume.py` を作成。`ndb_opendata_loader` のロジックを流用しつつ、特定薬効に絞って抽出・集計する。
2.  **国勢調査結合**: `02_load_census_data.py` で独居率などを計算し、BZデータと結合する。

### Phase 2: 記述統計・可視化
3.  **SPR算出**: `03_calculate_spr.py` で、高齢者1人あたりのBZ処方数量（Standardized Prescription Ratio like）を算出。
4.  **地図化**: 都道府県別のBZ処方量マップと、独居率マップを作成し、視覚的な類似性を確認する。

### Phase 3: 統計解析
5.  **空間的自己相関**: Global Moran's I を計算し、BZ処方の地域集積性を検定。
6.  **GWR (地理的加重回帰)**:
    - 目的変数: 高齢者1人あたりBZ処方量
    - 説明変数: 独居老人率、完全失業率、(調整変数: 精神科医師数)
    - 地域の経済状況（`NDB_XXX`で作ったデータ）も交絡因子として投入可能。
